# 数据源配置功能 - 快速参考

## ✨ 新功能

WebUI 现在支持灵活配置数据源！

## 🎯 支持的数据源类型

| 类型 | 说明 | 使用场景 |
|------|------|----------|
| **JSON 文件** | 本地 JSON 文件 | 开发测试、静态数据 |
| **JSON URL** | 远程 JSON API | 动态数据、第三方 API |
| **SQLite API** | SQLite 后端服务 | 完整的数据管理系统 |

## 🚀 快速开始

### 1. 打开配置

点击右上角 **"配置"** 按钮

### 2. 选择数据源

#### JSON 文件（默认）
```
文件路径: /data/data.json
```

#### JSON URL
```
URL: https://api.example.com/data.json
请求头: {"Authorization": "Bearer token"}
```

#### SQLite API
```
API 地址: http://localhost
端口: 3000
数据库: my-database
```

### 3. 测试并保存

1. 点击 **"测试连接"**
2. 确认成功后点击 **"保存配置"**

## 📝 数据格式

```json
[
  {
    "tags": ["标签1", "标签2"],
    "data": ["数据1", "数据2"]
  }
]
```

## 🔧 常用操作

- **刷新数据**: 点击右上角 **"刷新"** 按钮
- **切换数据源**: 打开配置 → 选择类型 → 保存
- **查看状态**: 右上角显示当前数据源类型

## 💡 提示

- 配置自动保存到浏览器
- 支持自定义请求头
- 支持 CORS 跨域请求
- 实时错误提示
