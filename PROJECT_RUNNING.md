## 🎯 新功能

### 数据源配置系统

在右上角你会看到：

1. **数据源显示** - 显示当前使用的数据源类型
2. **刷新按钮** - 重新加载数据
3. **配置按钮** - 打开数据源配置界面

### 支持的数据源

| 类型 | 状态 | 说明 |
|------|------|------|
| JSON 文件 | ✅ 可用 | 默认数据源，无需配置 |
| JSON URL | ✅ 可用 | 从远程 URL 加载数据 |
| SQLite API | ⚠️ 需要后端 | 需要启动 API 服务器 |

## 📝 快速开始

### 1. 访问应用

在浏览器中打开任一地址：
- http://127.0.0.1:5173/
- http://192.168.0.172:5173/

### 2. 测试数据源配置

1. 点击右上角 **"配置"** 按钮
2. 查看当前配置（默认：JSON 文件）
3. 可以切换到其他数据源类型
4. 点击 **"测试连接"** 验证配置
5. 点击 **"保存配置"** 应用更改

### 3. 使用标签过滤

1. 在标签选择器中选择标签
2. 查看匹配的数据
3. 可以选择多个标签进行组合过滤
4. 点击已选标签的 ❌ 可以移除

```bash

```

### 数据格式

```json

  {
```
