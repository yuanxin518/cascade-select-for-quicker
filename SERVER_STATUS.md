## 🔍 故障排除

### 如果无法访问 localhost:5173

这可能是因为你在 WSL2 环境中运行。请尝试：

1. **使用 127.0.0.1 代替 localhost**
   ```
   http://127.0.0.1:5173/
   ```

2. **使用 Windows 主机访问 WSL2 服务**
   - 在 Windows 浏览器中访问: http://192.168.0.172:5173/
   - 或使用 WSL2 的 IP 地址

3. **查找 WSL2 IP 地址**
   ```bash
   ip addr show eth0 | grep inet
   ```

### 查看服务器日志

```bash
tail -f /tmp/vite.log
```

### 停止服务器

```bash
pkill -f "vite"
```

### 重启服务器

```bash
cd /home/hyx_minipc/code/playground/quicker-tree-select
pnpm run dev:ui
```
